# CodeAlpha_Tasks

## Overview

This repository contains the projects completed during my CodeAlpha Artificial Intelligence Internship.

## Project 1: Language Translation Tool

* Developed a multilingual language translation application using Python and Streamlit.
* Enables users to translate text between multiple languages through a simple user interface.

## Project 2: FAQ Chatbot

* Built an AI-powered FAQ chatbot using Natural Language Processing (NLP).
* Uses TF-IDF Vectorization and Cosine Similarity to provide relevant answers to user queries.

## Project 3: Object Detection and Tracking

* Developed a real-time object detection and tracking system using YOLOv8 and OpenCV.
* Detects and tracks objects from live webcam video streams.

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* Natural Language Processing (NLP)
* OpenCV
* YOLOv8
* GitHub
